# Solitaire

Geboren uit het gemis van het spel dat al jaren niet meer standaard op Windows is geinstalleerd en omdat je toch iets te doen moet hebben in de trein als je geen internet hebt.

Begonnen in 2020 met Java en JavaFX, en daarna verder gegaan met Angular.
